# portal-main-pages

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态文件形式存放，不绑定任何特定域名或网站，主要服务于页面存档、结构展示与内容分发等通用用途。

## 项目简介

portal-main-pages 是一个轻量级的 HTML 页面集合仓库。每个页面均为独立的静态文件，包含各自的前端资源（如 CSS、JavaScript 及图片等）。仓库本身不包含后端逻辑，也不针对任何具体业务或网站进行定制。

## 目录说明

仓库根目录下按功能或用途划分若干子文件夹，每个子文件夹存放一类相关的 HTML 页面及其附属资源。常见的目录结构示例如下：

```
portal-main-pages/
├── page-a/          # 页面 A 相关文件
│   ├── index.html
│   ├── style.css
│   └── script.js
├── page-b/          # 页面 B 相关文件
│   ├── index.html
│   └── assets/
└── README.md        # 本文件
```

具体目录的名称和内容随仓库更新而调整。

## 页面归档说明

- 所有页面均为静态 HTML 文件，可直接通过浏览器打开或部署至任意静态托管服务。
- 页面中可能引用外部资源（如字体、图标库），但尽量保持自包含，减少外部依赖。
- 部分页面可能包含示例数据或占位内容，仅供演示或归档用途。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器打开任意子目录下的 `index.html` 文件即可预览。
3. 也可将整个仓库部署至静态托管平台，通过 URL 访问对应页面。
